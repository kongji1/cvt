CPA / sub2api ChatGPT Session

ChatGPT JSON / Session 本地转换器

单文件离线工具。支持 JSON、JSONL、ChatGPT Web session、Codex auth.json、9router OAuth 和 sub2api bundle；全部在浏览器本地转换，不上传内容。

仅转换你有权使用的登录信息。Web session 通常没有 refresh_token，access_token 过期后不能自动刷新。

从当前已登录的 ChatGPT 导入

登录 ChatGPT 后，打开 https://chatgpt.com/api/auth/session，复制整段 JSON 再导入。

打开 Session JSON
  1. 点击「打开 Session JSON」；若已登录，该页会显示 Session JSON。
  2. 在该页复制全部内容，回到本页点「从剪贴板导入并转换」。

也支持 AT-only JSON（type、access_token、refresh_token）以及卡密导出 TXT：首行为「卡密导出」或含 === 卡密内容 ===，后续每行一个 AT-only / sub2api JSON。

浏览器同源隔离不允许本站读取 ChatGPT 的登录 Cookie 或自动复制 Session；上述流程中的复制一步必不可少。本页仅在本地读取剪贴板并转换。
